Funebra Math Helpers

Reference index of helper functions for the Funebra Math-Art Engine.

Available Helpers

These helpers return coordinates for parametric plotting and math-art geometry:

  • polygonY(o, sides, radius, centerY, stepsPerEdge)
  • polygonX(o, sides, radius, centerX, stepsPerEdge)
  • starX(o, arms, radius, centerX)
  • starY(o, arms, radius, centerY)
  • crossX(...)
  • crossY(...)
  • crossZ(...)
  • pyramidX(...)
  • pyramidY(...)
  • pyramidZ(...)
  • heart2D(...)
  • triX(...)
  • triY(...)
  • cube(...)

Usage

These helpers can be combined to generate coordinates for visual math experiments, geometry art, and 3D exports. Example: 


const sides = 6, R = 140, stepsPerEdge = 40;
const CX = 360, CY = 220;

function X(o){ return polygonX(o, sides, R, CX, stepsPerEdge); }
function Y(o){ return polygonY(o, sides, R, CY, stepsPerEdge); }
