How to read this like FL Studio:
▸ Horizontal = time: 40 beat-cells (4 beats × 10 bars).
▸ Vertical = pitch lanes: higher rows ≈ higher notes.
▸ Note length is the number of beat-cells it spans (1 = quarter, 2 = half, 4 = whole bar).
▸ Verse (bars 1–8) uses G – Em – C – D twice, chords as long blocks, melody as shorter phrases.
▸ Bars 9–10 hint the chorus, tightening the chords and making the melody more active.
▸ In FL Studio, set tempo ≈ 90 BPM, drop these chords in the piano roll, then copy the relative melody timing by eye.
